Table des Transformées de Laplace

Exemple détaillé : F(p) = 1/(p²+4)

Pour trouver l'original de \[ F(p) = \frac{1}{p^2+4} \]

1) On reconnaît la forme \( \frac{1}{p^2+a^2} \) avec \( a^2 = 4 \) donc \( a = 2 \)

2) D'après la table, \( \frac{a}{p^2+a^2} \) est la transformée de \( \sin(at) \)

3) Donc \( \frac{1}{p^2+4} = \frac{1}{2} \cdot \frac{2}{p^2+4} \)

4) L'original est : \[ f(t) = \frac{1}{2}\sin(2t), \quad t \geq 0 \]

f(t) Fonction originale F(p) Transformée de Laplace Exemple
\[ f(t) = 1 \] \[ F(p) = \frac{1}{p} \] Pour \(f(t) = 3\): \[ F(p) = \frac{3}{p} \]
\[ f(t) = t \] \[ F(p) = \frac{1}{p^2} \] Pour \(f(t) = 2t\): \[ F(p) = \frac{2}{p^2} \]
\[ f(t) = t^n \] \[ F(p) = \frac{n!}{p^{n+1}} \] Pour \(f(t) = t^3\): \[ F(p) = \frac{6}{p^4} \]
\[ f(t) = e^{at} \] \[ F(p) = \frac{1}{p-a} \] Pour \(f(t) = e^{3t}\): \[ F(p) = \frac{1}{p-3} \]
\[ f(t) = \sin(at) \] \[ F(p) = \frac{a}{p^2+a^2} \] Pour \(f(t) = \sin(2t)\): \[ F(p) = \frac{2}{p^2+4} \]
\[ f(t) = \cos(at) \] \[ F(p) = \frac{p}{p^2+a^2} \] Pour \(f(t) = \cos(3t)\): \[ F(p) = \frac{p}{p^2+9} \]
\[ f(t) = t\sin(at) \] \[ F(p) = \frac{2ap}{(p^2+a^2)^2} \] Pour \(f(t) = t\sin(2t)\): \[ F(p) = \frac{4p}{(p^2+4)^2} \]
\[ f(t) = t\cos(at) \] \[ F(p) = \frac{p^2-a^2}{(p^2+a^2)^2} \] Pour \(f(t) = t\cos(2t)\): \[ F(p) = \frac{p^2-4}{(p^2+4)^2} \]
\[ f(t) = te^{at} \] \[ F(p) = \frac{1}{(p-a)^2} \] Pour \(f(t) = te^{2t}\): \[ F(p) = \frac{1}{(p-2)^2} \]
\[ f(t) = t^2e^{at} \] \[ F(p) = \frac{2}{(p-a)^3} \] Pour \(f(t) = t^2e^{3t}\): \[ F(p) = \frac{2}{(p-3)^3} \]
\[ f(t) = \sinh(at) \] \[ F(p) = \frac{a}{p^2-a^2} \] Pour \(f(t) = \sinh(4t)\): \[ F(p) = \frac{4}{p^2-16} \]
\[ f(t) = \cosh(at) \] \[ F(p) = \frac{p}{p^2-a^2} \] Pour \(f(t) = \cosh(2t)\): \[ F(p) = \frac{p}{p^2-4} \]
\[ f(t) = t^n e^{at} \] \[ F(p) = \frac{n!}{(p-a)^{n+1}} \] Pour \(f(t) = t^2e^t\): \[ F(p) = \frac{2}{(p-1)^3} \]

Propriétés importantes

1. Linéarité : \[ \mathcal{L}\{af(t) + bg(t)\} = aF(p) + bG(p) \]

2. Translation temporelle : \[ \mathcal{L}\{f(t-a)u(t-a)\} = e^{-ap}F(p) \]

3. Translation fréquentielle : \[ \mathcal{L}\{e^{at}f(t)\} = F(p-a) \]

4. Dérivation : \[ \mathcal{L}\{f'(t)\} = pF(p) - f(0) \]

5. Dérivée n-ième : \[ \mathcal{L}\{f^{(n)}(t)\} = p^nF(p) - p^{n-1}f(0) - p^{n-2}f'(0) - ... - f^{(n-1)}(0) \]

6. Intégration : \[ \mathcal{L}\{\int_0^t f(\tau)d\tau\} = \frac{F(p)}{p} \]
